# viper2
A simple WhatsApp bot to manage groups

## Deploy

You can deploy this bot to Heroku using the button below. It will create a new app from this repository and let you set environment variables (like `SESSION_ID`) during the deploy process.

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/ARNOLDT20/viper2)

Notes:
- Make sure to add `SESSION_ID` and any other required environment variables when prompted by Heroku.
- This repo contains session-related defaults in `config.js`; do not commit real session tokens to the repository.
